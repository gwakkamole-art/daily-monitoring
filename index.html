📂 새 채널 그룹 추가

YouTube API 초간단 발급 가이드 뽀바박!

1 구글 클라우드 콘솔 접속 및 로그인
[여기 클릭] 해서 사이트로 이동한 뒤 구글 계정으로 로그인하세요.
2 프로젝트 만들기 (중요!)
왼쪽 위 로고 옆의 [프로젝트 선택] 또는 [My First Project]를 누르고, 팝업창 오른쪽 위의 [새 프로젝트]를 클릭해 이름을 대충 정하고 만드세요.
3 YouTube API 활성화 (가장 많이 빼먹음!)
상단 검색창에 'YouTube Data API v3'를 입력해 검색합니다.
나오는 화면에서 파란색 [사용(Enable)] 버튼을 반드시 클릭해야 합니다!
4 API 키 생성
왼쪽 메뉴 [사용자 인증 정보] → 상단 [+ 사용자 인증 정보 만들기] → [API 키]를 누르면 끝!
AIza...로 시작하는 키를 복사하세요.

동료분들을 위한 쪼수의 최종 체크리스트

  • 상단 프로젝트 이름을 확인했나요?
    내가 방금 만든 프로젝트가 선택되어 있어야 합니다.
  • [사용(Enable)] 버튼을 진짜 누르셨나요?
    이걸 안 누르면 키를 넣어도 "오류 발생"만 뜹니다!
  • API 이름이 'v3'가 맞나요?
    반드시 YouTube Data API v3여야 합니다.
