YouTube Data API v3 발급 방법
1️⃣ Google Cloud Console 접속 후 새 프로젝트를 만듭니다.

3️⃣ 왼쪽 메뉴 [사용자 인증 정보] → [+ 사용자 인증 정보 만들기] → [API 키] 클릭!
